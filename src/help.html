COMPA - Basic toolbar

Pan
Pan map by dragging
Zoom In
Zoom in map by draw extent
Zoom Out
Zoom out map by draw extent
Identify
Click feature in map to display attributes
(layer options are same as Select tool)
Select
Select features in map within draw extent (mark as yellow)
Layer option
- Top Layer: Select features in top-most layer only
- Visible Layers: Select features in visible layers only
- All Layers: Select features in all layers of map
Clear Select
Clear selected feature
Full Extent
Zoom to overall view of map
Scale
Show/Set scale of map
Back Extent
Go to previous map view
Next Extent
Go to next map view
Basemap
Change basemap of map
Swipe
Chose a layer in menu then press and drag on the map to swipe
(all layers of map list in menu)
Transparent
Make/unmake a layer of map transparent
(usefull when editing)
Measure
Measure on map (turn on snapping for accurate measure)
- Location: Determine x, y map coordinates at point click
- Length: Measure length by draw polyline on map
- Area: Measure area by draw polygon on map
Print
Print current map view
(chose page template in menu)
Help
Show user manual document

COMPA - Edit toolbar

Edit Menu
Edit menu
- Start editing: Check-out selected features and begin new edit-trasaction
- Stop editing: Stop current edit-trasaction
- Save edits: Save edits in current trasaction to server
- Start editing (session)*: Begin new edit-trasaction in longtime session
- Show deleted features*: Show/hide deleted features of current longtime session

check-out 5 features editing
Edit Select
Selection tool in edit mode (mark as green)

(click to select only one feature, see Select tool for layer options)
Add to Edit
Add selected features to current edit-transaction

Attribute
Open attribute table for edit (double click a cell to edit value)
Property
Open property dialog for edit (select features to edit property - using widget AttributeInspector)
Pen
Pen tool use to create new feature (select template before create new feature - using widget TemplatePicker)

click on map to draw, double-click to finish
Auto-complete Polygon
Create new polygon that adjacent (auto-complete) to exist polygons (draw a polyline has start & end point touch exist polygons)
Move/Rotate/Scale
Move, rotate and scale selected feature (can click on selected feature to activate)

use anchor point to manipulate
Edit Vertices
Edit each vertices of feature can double-click on selected feature to activate

drag vertex to move, drag ghost-vertex to add new vertex
Duplicate
Duplicate selected features
Delete
Delete selected features
Buffer
Buffer selected feature by distance (negative value to narrow)
Merge
Merge adjacent selected features
Split
Split selected features by a cut-polyline (turn on snapping for accurate measure)
Reshape
Reshape selected feature by a edge-polyline (draw a polyline that touch selected feature)

turn on snapping for accurate measure
Undo
Undo recent edit
Redo
Redo recent undo-edit

COMPA - Snap toolbar

Snap Settings
Settings snap environment
- Tolerence: Tolerence distance of snap
- Activate: Activate snap always/pressing Ctrl
- Style: Shape of snap point
- Color: Color of snap point
Enable Snap
Check/uncheck to turn on/off snapping
Snap Point
Snap to point yes/no
Snap Vertex
Snap to vertex yes/no
Snap Edge
Snap to edge yes/no
